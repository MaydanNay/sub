Мобильное  приложение  для  проведения  виртуальных  беговых
марафонов , позволяющее  участникам  бегать  где  угодно  и  когда
удобно , сохраняя  атмосферу  и  механику  традиционных  офлайн -
марафонов [1][2]. Проект  переносит  опыт  массовых  забегов  в
цифровую  среду , расширяя  географический  охват  и  устраняя
логистические  ограничения  ( перекрытие  дорог , привязка  к
одному  городу )[1].
Создание  платформы , где  участники :
Регистрируются  на  марафоны  с  оплатой  взноса
Выбирают  удобное  время  и  место  для  забега
Получают  физические  медали  и  мерч
Участвуют  в  рейтингах  и  получают  призы
Взаимодействуют  через  геймификацию

Анализ  успешных  беговых  приложений  выявил  несколько
интересных  механик , которые  значительно  повышают
вовлечённость  пользователей .
Описание  механики[6][7][8]:
Приложения  INTVL и  Stride превращают  город  в  игровое  поле ,
разделяя  карту  на  участки [6][8]. Когда  бегун  пробегает  по
определенной  территории , он  " захватывает " её  и  закрепляет  за
собой [6]. Другие  пользователи  могут  " атаковать " чужие
территории , пробегая  по  ним  чаще  — тогда  зона  меняет  цвет  ( с
зеленого  на  желтый  при  атаке )[6]. Игроки  могут  объединяться  в
команды  для  совместной  защиты  и  захвата  районов [6][7].
Применение  в  проекте  ShuRun:
Преимущества  для  вовлечённости:
Мотивация  исследовать  новые  маршруты
Соревновательный  элемент  между  пользователями
Возвращаемость  в  приложение  ( защита  территорий )
Командная  механика  для  социального  взаимодействия
3.3 Геймификация  и  уникальные  механики
3.3.1 Захват  территорий  ( аналог  INTVL , Stride)
Карта  города  делится  на  зоны  ( районы , кварталы ,
микрорайоны )
•
Пробежка  по  зоне  закрепляет  её  за  пользователем  ( первое
прохождение )
•
Система  " защиты " территории  — нужно  регулярно
пробегать  для  удержания
•
Рейтинг  по  количеству  контролируемой  территории
(квадратные  километры )
•
Командные  режимы  ( борьба  команд  за  районы  города )•
Уведомления  при  " атаке " на  твою  территорию [9]•
Визуализация  на  карте  ( цветовое  кодирование  зон  по
владельцам )
•
Бонусы  за  удержание  территорий  ( опыт , достижения )•

Описание  механики[10][11]:
GPS- арт  — это  создание  рисунков  через  маршруты  пробежек [10]
[11]. Бегуны  заранее  планируют  маршрут , который
складывается  в  картинку  ( животное , символ , слово ), и  проходят
его  с  включенным  GPS- трекингом [11]. В  России  уже  существует
проект  «GPS арт » с  форматами  СИТИ  ( городские  дорожки ) и
ТРЕЙЛ  ( вне  дорожек )[10].
Примеры  популярных  рисунков : животные  ( заяц , медведь ,
птица ), символы  ( сердце , звезда ), слова  и  поздравления [11].
Применение  в  проекте  ShuRun:
Преимущества  для  вовлечённости:
Творческий  подход  к  бегу  ( не  просто  километры , а
искусство )
Виральность  ( пользователи  делятся  красивыми
маршрутами )
Исследование  города  через  необычные  маршруты
Уникальный  контент  для  каждого  города
3.3.2 GPS- арт  ( рисование  маршрутами )
Библиотека  готовых  маршрутов - рисунков для  разных
городов
•
Конструктор  маршрутов — инструмент  для  создания
собственных  арт - маршрутов
•
Челленджи — " нарисуй  животное / символ  за  неделю "•
Галерея  GPS- артов от  сообщества  с  лайками  и
комментариями
•
Рейтинг  самых  креативных  маршрутов (голосование
сообщества )
•
