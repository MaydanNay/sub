# Структура проекта Gamification Catalog

Этот файл описывает организацию файлов в репозитории для понимания структуры проекта.

## Корневая директория
- `/backend` — Исходный код бэкенда (FastAPI).
- `/frontend` — Исходный код фронтенда (React + Vite).
- `/docs` — Техническая документация.
- `sql_app.db` — База данных SQLite (создается при первом запуске).
- `run_project.py` / `start_servers.bat` — Скрипты для локального запуска разработки.

## Backend (/backend)
- `main.py` — Точка входа в приложение, настройка FastAPI и роутов.
- `database.py` — Настройка подключения к SQLAlchemy (SQLite).
- `models.py` — Определение таблиц базы данных.
- `schemas.py` — Pydantic-модели для валидации данных API.
- `crud.py` — Логика работы с базой данных (Create, Read, Update, Delete).
- `requirements.txt` — Список зависимостей Python.

## Frontend (/frontend)
- `/src` — Исходный код React.
  - `/components` — Общие UI компоненты и иконки.
  - `/pages` — Страницы каталога и игровые экраны.
  - `/data` — Маркетинговый контент и настройки игр.
  - `/utils` — Вспомогательные скрипты (звук и т.д.).
- `/public` — Статические ассеты.
- `package.json` — Зависимости Node.js и скрипты сборки.
- `vite.config.js` — Конфигурация сборщика Vite.
